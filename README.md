# Strictly Business

A private, members-only recommendation directory for the Strictly Business group —
eight years of "who do you recommend for…?" turned into a ranked, searchable list.

The directory data is **AES-256 encrypted**. Open the app and enter the group
gematria to unlock it; decryption happens entirely in your browser. Add it to your
home screen to use it like an app.

No plaintext names or numbers are stored in this repository.
